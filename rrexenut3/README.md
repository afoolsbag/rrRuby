# 运动与营养（Exercise and Nutrition）第三版

为记录身体、运动与营养数据，指导塑形健身而作。

*   `v3`：CLI，使用 Ruby 编写
*   `v2`：GUI，使用 JavaScript 编写，过于笨重
*   `v1`：GUI，使用 C++ 编写，过于笨重
